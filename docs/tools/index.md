title: Tool Reference
---

`mcp-server-template` provides a compact starter toolset that demonstrates recommended MCP tool patterns.

Most tools accept `response_format`: `"markdown"` (default) or `"json"`.

## Tool Groups

- [Utility Tools](/tools/utility)
  - Tools: `template_ping`, `echo`, `text_transform`, `current_time`, `system_info`
  - Purpose: read-only diagnostics and text utilities

## Common Parameters

These parameters are shared across most tools:

- `response_format` (`"markdown" | "json"`): output format (defaults to `"markdown"`).

## Quick Reference

See [Utility Tools](/tools/utility) for detailed inputs and examples.
