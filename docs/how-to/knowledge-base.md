title: Knowledge Base Authoring
---

## Use this guide when

You need to search, create, or update TeamDynamix KB content.

## Steps

1. Resolve KB `app_id` from `teamdynamix_list_applications`
2. List categories using `teamdynamix_list_kb_categories`
3. Search existing content using `teamdynamix_search_kb_articles`
4. Create article using `teamdynamix_create_kb_article` (write enabled)
5. Update article using `teamdynamix_update_kb_article` (write enabled)
6. Verify with `teamdynamix_get_kb_article`

## Practical tips

- Keep category selection explicit to avoid misclassification
- Always search first to reduce duplicate articles
